Multiple values

It's possible to ignore more than one category at a given time by using a set of comma seperated values. 


  {% postlist ignore_category='event,colors,self improvement' %}

Category ommission

When ignore_category's value is empty then the database instruction to ignore categories is skipped.


  {% postlist ignore_category='' %}

This is useful when using a template variable and the variable does not have a known or assigned value yet.


  {% postlist ignore_category='{{ MY_VAR }}' %}
