The bloglist supports organising featured posts. When marking a bloglist as featured, this results in pinning featured posts at the top of the list, making it more visible.

Moreover, featured posts gets additional CSS styling and differentiates each post -- meaning that featured posts have unique styling compared with normal posts.

Note

Extra CSS has been applied to this demo. Notably, the "featured" block is intended for presentation.

{% bloglist featured='True' %}

Enabling featured properties is directly tied to both the model's feature attribute and a bloglist count. The default featured count is 1. For example, the post titled "Mazda RX-7" is actually inside the database under featured, however it's position in this example was greater than the count, therefore anything greater skips the featured properties.

Using filters and featured


  {% bloglist featured='True' category='colors' %}

The formula to gather featured posts is calculated by the current filter being applied to the bloglist. Since a bloglist can be filtered by date or category, the associated feature will be similar.
