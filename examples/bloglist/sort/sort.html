The sort key changes the order which the posts are presented.

Sort is a choice-based option, the options are:

  1. asc - stands for ascending (a-z)
  2. desc - stands for descending (z-a)

These options are not case sensitive with uppercase and lowercase both considered equal, such as "DESC" vs "desc" - gives the same result.

Standard sorting

When the "sort" option is discluded then the posts are arranged by publish dates, with the newest post appearing at the top.

{% bloglist %}

Sort by ascending

Sort by ascending means arranging items from the smallest or lowest value to the largest or highest value.
