## Welcome

Django Blog Improved (DJ BIM for short) is blog publishing software built with [Django](https://www.djangoproject.com/). DJ BIM assists you in becoming a blog owner with fewer hurdles. 

DJ BIM comes with publishing components. Each component is a tool that covers tasks common to the blogging sphere.
